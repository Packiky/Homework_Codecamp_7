Sequence Question Answer
4.1 value ใน display property มีอะไรบ้าง
  • inline : Displays an element as an inline element (like span). Any height and width properties will have no effect
  • block : Displays an element as a block element (like p). It starts on a new line, and takes up the whole width
  • contents : Makes the container disappear, making the child elements children of the element the next level up in the DOM
  • flex : Displays an element as a block-level flex container
  • grid : Displays an element as a block-level grid container
  • run-in : Displays an element as either block or inline, depending on context
4.2 grid และ flex แตกต่างกันอย่างไร

Flex ใช้เพื่อจัดการ layout 1D ส่วน Grid ใช้เพื่อจัดการ layout 2D.

Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน

Grid จัดวางตรงไหนก็ได้
4.3 justify and align แตกต่างกันอย่างไร

justify คือการจัด content ตาม Main Axis

align คือ การจัดcontent ตามแนว Cross Axis และ items ต้องมากกว่า 1 บรรทัดถึงจะทำงาน

5.1 position คืออะไร และ อะไรบ้าง

position คือ property ที่ใช้ในการกำหนดตำแหน่งของ element

  1. static : default value
  2. relative :เป็นการกำหนดตำแหน่งนับจาก element ที่อยู่ก่อนหน้าโดยใช้ top left bottom right
  3. absolute : กำหนดตำแหน่งของ element ด้วย top left bottom right (absolute จะไม่สนใจ element ก่อนหน้า และจะพาตัวเองไปยังจุดนั้นเลย)
  4. fixed : ตรึง element นั้นให้อยู่ที่เดิมของหน้าจอตลอดไม่ว่าจะเลื่อนไปไหน
  5. sticky : เป็นการวางตำเเหน่งที่จะยังติดไปเเม้มีการเลื่อนหน้า
5.2 z-index คืออะไรใช้อย่างไร

คือ การจัดการ element ในรูปแบบของ layer

ค่าเริ่มต้น z-index = 0

ค่าน้อย(ติดลบ) จะอยู่ด้านหลัง, ค่ามาก จะอยู่ด้านหน้า
