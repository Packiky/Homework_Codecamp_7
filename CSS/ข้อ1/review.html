1.4 combinator มีกี่แบบอะไรบ้าง

4 แบบ

  1. Descendant selector (space) : matches all elements that are descendants of a specified element
  2. Child selector (>) : selects all elements that are the children of a specified element.
  3. Adjacent sibling selector (+) : selects all elements that are the adjacent siblings of a specified element.
  4. General sibling selector (~) : selects all elements that are siblings of a specified element.
